Negrito e Itálico Semânticos

Empregando o Negrito

Nesta frase vou utilizar o negrito de uma forma não semântica com a TAG <b>. Não se deve fazê-lo

Nesta frase vou utilizar o negrito de uma forma semântica. Deve-se fazê-lo com a tag <strong>

Empregando o itálico

Nesta frase vou utilizar o itálico de uma forma não semântica com a TAG <i>. Não se deve fazê-lo

Nesta frase vou utilizar o itálico de uma forma semântica. Deve-se fazê-lo com a tag <em>
