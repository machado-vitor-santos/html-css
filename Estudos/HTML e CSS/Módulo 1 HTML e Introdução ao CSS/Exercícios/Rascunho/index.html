Para deixar em negrito de forma semântica Usa-se o <strong> , por exemplo: deixarei esta frase em negrito

para deixar uma frase em itálico usa-se o <em> , por exemplo: deixarei esta frase em itálico

Para marcar o texto, utiliza-se o <mark>, segue esta frase em evidência

Para deizar o texto deletado, utiliza-se o <del>, por exemplo esquça isso

Para deixar um texto menor utiliza-se o <small>, dificil de ler pequeno
